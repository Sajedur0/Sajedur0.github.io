# Sajedur0 Portfolio

A personal portfolio website for **MD Sajedur Rahman Roni** — a software & hardware enthusiast from Narsingdi, Bangladesh. Built with vanilla HTML, CSS, and JavaScript featuring a modern neumorphic (soft UI) design.

**Live:** [sajedur0.github.io](https://sajedur0.github.io)

---

## Features

- **Neumorphic Design** — Soft UI with extruded and inset shadows throughout
- **Dynamic GitHub Projects** — Repos are fetched in real-time from a backend API and displayed as cards
- **Server-Sent Events (SSE)** — Projects update live without page refresh
- **Responsive Layout** — Fully responsive from 360px to 1440px+ with a mobile hamburger menu
- **Scroll Reveal Animations** — Cards fade up as they enter the viewport via IntersectionObserver
- **Smooth Scrolling** — Clean anchor navigation without hash in the URL
- **Custom 404 Page** — Redirects unknown paths back to the correct section
- **Reduced Motion Support** — Respects `prefers-reduced-motion` for accessibility

## Tech Stack

- **HTML5** — Semantic markup
- **CSS3** — Custom properties, CSS Grid, Flexbox, neumorphic shadows
- **JavaScript (ES5+)** — Vanilla JS, no frameworks
- **Google Fonts** — Space Grotesk, IBM Plex Sans, IBM Plex Mono
- **Backend API** — (Vercel) for GitHub repo data via REST + SSE
